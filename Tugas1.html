\[Biner\]

atribut yang hanya memiliki 2 nilai mutlak, seperti 0 dan 1.

Atribut binary dapat dibagi lagi dalam dua jenis yakni: teks tebal

  1. Simetris: Kedua nilai sama derajat gunanya (misal: gender: laki-laki & perempuan).

  2. Asimetris: Kedua nilai tidak sama derajat gunanya (misal: hasil: lulus & gagal). Operasi yang dapat dilakukan distinctness.daftar

\[Perhitungan Biner(Simetris)\]
\[Rumus: d(i,j) = \ \frac{r+s} {q+r+s+t} \]
\[Keterangan: \]
                                                     *   r: i=1, j=0
                                                     *   s: i=0, j=1 
                                                     *   q: i=1, j=1
                                                     *   t: i=0, j=0 

def binersimetris(i,j):
  q=r=s=t=0
  for col in ['gender','test preparation course']:
    if X[col][i]==1 and X[col][j] == 1:
      q+=1
      print("nilai q =",q)
    elif X[col][i]==1 and X[col][j]==0:
      r+=1
      print("nilai r =",r)
    elif X[col][i]==0 and X[col][j] == 1:
      s+=1
      print("nilai s =",s)
    else :
      t+=1
      print("nilai t =",t)
  print("Rumus = (r+s)/(q+r+s+t)")
  return (r+s)/(q+r+s+t)

print("Nilai Biner = ", binersimetris(1,3))

\[Nominal\]

Nominal Nominal adalah jenis atribut yang hanya menyediakan nilai yang cukup untuk membedakan antara satu objek dengan objek lainnya. Nilai dari atribut nominal adalah nama benda atau dapat juga berupa simbol. Jenis atribut ini hanya bisa dilakukan operasi membandingkan sama atau tidaknya suatu data.

Contoh : nama orang, alamat, nomor ktp, kode pos, warna mata dan lain-lain.

Operasi yang bisa dilakukan adalah distinctness.

\[Perhitungan Nominal\]
\[ Rumus: d(i,j) = \frac{m-p}{p}\]
\[Keterangan : \]
                      *   m = jumlah data yang dibandingkan
                      *   p = jumlah data sama

def d1(i,j):
    P = 4
    M = 0
    for col in ["race/ethnicity","parental level of education"]:
        if X[col][i] == X[col][j]:
            M += 1

    return (P-M)/P

import math

def d2(x,y,h):
    result = 0
    for col in [hasil1, hasil2, hasil3]:
        result = math.pow(abs(col[x] - col[y]), h)
    return result**(1/h)
